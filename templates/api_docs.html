Mobile REST Integration

AgriVision AI has been engineered with a mobile-first philosophy. All diagnostic logic, computer vision matrices, and fertilizer recommendation charts are accessible via lightweight JSON REST APIs. This allows Flutter, React Native, or Native Android developers to construct highly responsive field apps for farmers.

1. Image Diagnostic Endpoint

POST /api/scan

Upload leaf image for disease diagnosis and color ratio metric evaluation.

Request Type: multipart/form-data

Key Type Required Description
image File (binary) Yes Leaf photo (JPG, PNG)
crop String No coconut or arecanut (default: coconut)

Mobile Code SDK Snippets

Use the http package to send leaf photos to the Flask backend from your Flutter app: 

import 'dart:io';
import 'package:http/http.dart' as http;
import 'dart:convert';

Future<Map<String, dynamic>> diagnoseLeaf(File imageFile, String crop) async {
  var request = http.MultipartRequest(
    'POST', 
    Uri.parse('http://YOUR_SERVER_IP:5000/api/scan')
  );
  
  // Attach variables
  request.fields['crop'] = crop;
  request.files.add(
    await http.MultipartFile.fromPath('image', imageFile.path)
  );

  var response = await request.send();
  if (response.statusCode == 200) {
    var responseData = await response.stream.bytesToString();
    return jsonDecode(responseData);
  } else {
    throw Exception('Failed to diagnose leaf.');
  }
}

Use the fetch API to upload files using FormData in React Native: 

const diagnoseLeafImage = async (imageUri, cropType = 'coconut') => {
  const formData = new FormData();
  formData.append('crop', cropType);
  formData.append('image', {
    uri: imageUri,
    name: 'leaf_photo.jpg',
    type: 'image/jpeg',
  });

  try {
    const response = await fetch('http://YOUR_SERVER_IP:5000/api/scan', {
      method: 'POST',
      body: formData,
      headers: {
        'Content-Type': 'multipart/form-data',
      },
    });
    const result = await response.json();
    return result;
  } catch (error) {
    console.error("API Diagnostic Error: ", error);
  }
};
